# 🤖 CodeAlpha Artificial Intelligence Chatbot (Java)

## 🚀 Project Overview

This project is a simple Artificial Intelligence Chatbot developed using
Java. It interacts with users through the console and provides responses
based on keyword matching (basic NLP).

## ✨ Features

-   Interactive chatbot (console-based)
-   Keyword-based response system
-   Rule-based logic
-   Handles multiple user queries
-   Easy to extend with new questions and answers

## 🛠️ Technologies Used

-   Java
-   OOP Concepts
-   Basic NLP (Keyword Matching)
-   Conditional Statements

## 📂 Project Structure

com.chatbot │ └── ChatBot.java

## ▶️ How to Run

1.  Open project in Eclipse / IntelliJ
2.  Compile the program
3.  Run ChatBot.java
4.  Start chatting in the console

## 💻 Sample Output

ChatBot: Hello! Ask me something. You: hi ChatBot: Hello! How can I help
you?

You: what is java ChatBot: Java is a programming language.

You: exit ChatBot: Goodbye!
